# Handwritten Digit Recognition using Neural Network

This project is a Handwritten Digit Recognition system developed using Deep Learning and Artificial Neural Networks (ANN).  
The model is trained on the MNIST dataset to recognize digits from 0–9 with high accuracy.

---

## 📌 Project Overview

The main objective of this project is to classify handwritten digits using a neural network model.

The project includes:
- Data preprocessing
- Model training
- Prediction
- Accuracy evaluation
- Visualization of results

---

## 🛠 Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

Dataset Used: **MNIST Handwritten Digits Dataset**

The dataset contains:
- 60,000 training images
- 10,000 testing images
- 28x28 grayscale handwritten digit images

---

## 🧠 Deep Learning Model

Artificial Neural Network (ANN) model with:
- Input Layer
- Hidden Layers
- Output Layer

Activation Functions:
- ReLU
- Softmax

Loss Function:
- Sparse Categorical Crossentropy

Optimizer:
- Adam
